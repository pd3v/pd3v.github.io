Launch line

If line's folder in your Desktop

  • Open Terminal (MacOS) or similar

  • Type cd ~/Desktop/line/build {ENTER}

  • Type ./line {ENTER}

<<< line is now running >>>

Now run your favorite synth, sampler, DAW, or other MIDI receiver!

line v0.8 Manual

With line running, type:

Send a MIDI message:

4 1/4 midi notes

45 46 47 48 or a3 as3 b3 c4

1/4 1/8 1/8 1/4 1/4 notes

45 .34 35. 48 49 or a3 .as2 b2. c4 cs4

'-' for a rest note

36 .37 38. 41 .- 46.

C Major chord + C Major arpeggio. All 1/4

(c3 e3 g3) g5 e6 c2

d note with 0.5 of amplitude

d3~.5

C Minor chord with 0.7 of amplitude

(c4 eb4 g4)~.7

Note: amplitude is 1.0, by default

More on chords:v0.8.1

For major, minor, half-dimished, diminished chords, add 7th and 9th, line has chord cipher shortcuts.

D Minor minor 7th chord

(dminm7)

B Half-Diminished

(bhalfdim)

B Dim

(bdim)

G Dominant

(gmajm7) (g4majm7)

C# Major add 9

(cs2majM9)

Create a C major phrase with 8 notesv0.8.1

seq(8) major // resulting phrase: c d e f g a b MIDI equivalent: 0 2 4 5 7 9 11 0

Create a D scale phrase with 8 notesv0.8.1

seq(8) major raise(2) soct(4) // resulting phrase: d4 e4 f#4 g4 a4 b4 c#4 d4 MIDI equivalent: 50 52 54 55 57 59 61 50

Create a sequence of 16 notes, 2 octaves of C Major scalev0.8.1

seq(16) major(c4)

seq(16) major soct(4) // same as above

Create a 4 notes chord with the first 4 notes of D locrian scalev0.8.1

(_ _ _ _) locrian(d4)

Every command can be combined in one single line

Plays 4 different octaves of c, 4 of each, total 16 notes, all randomly positioned. mult and s will run once.

c4 c5 c6 c7 mult(4) s


"_" underscore char adjacent after any command

The same previous combo but randomly positioned every new cycle/bar, forever

c4 c5 c6 c7 mult(4) s_

The same previous combo but randomly positioned next 2 cycles/bars

c4 c5 c6 c7 mult(4) s_2
